# 팀글벙글
팀글벙글은 프로젝트 일정 관리를 효율적으로 할 수 있도록 도와주는 협업 툴입니다.   
다양한 기능을 통해 팀원 간의 소통과 작업 관리를 원활하게 진행할 수 있습니다. 

## 기술 스택
### 백엔드
- Spring boot
- Spring Security
- Redis
- Socket io
